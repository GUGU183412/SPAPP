# UI Mockups

This directory stores raw UI mockups shared during the design exploration phase.

Current files:

- `feedback-v1.html`: feedback page draft for resolved / unresolved response collection
- `landing-page-v1.html`: landing page draft for ASIN confirmation and tutorial entry
- `step1-v1.html`: initial Step 1 UI draft
- `step1-v2.html`: revised Step 1 UI draft focused on lock verification
- `step2-v1.html`: Step 2 UI draft focused on safety endorsement and stitching trust
- `step3-v1.html`: Step 3 UI draft focused on door protection and hinge-side placement
- `step3-v2.html`: revised Step 3 UI draft with stronger zero-damage messaging and clearer hinge/latch diagram
- `unresolved-reason-v2.html`: follow-up page draft for unresolved reason capture with recommended review guidance

These files are preserved as provided for review, comparison, and later implementation reference.
